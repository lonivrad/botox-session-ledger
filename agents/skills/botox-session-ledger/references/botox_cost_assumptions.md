# Botox Cost and Operational Assumptions

This file contains the default operational assumptions used by:

`scripts/botox_session_ledger.py`

These assumptions support:

- inventory reconciliation
- consumable cost estimates
- pricing recommendations
- gross margin estimates
- operational session auditing

This reference is informational only.

This skill does **not** recommend Botox dosing, injection sites, treatment plans, routes, frequency, or clinical appropriateness.

---

# Product assumptions

## Botox

- Product: Botox
- Manufacturer: Allergan
- Default vial size: **100 units**
- Default vial cost: **$656.00**
- Common real-world vial cost range: **$400–$700** depending on purchase volume

---

# Reconstitution assumptions

## Bacteriostatic saline

- Default saline cost per vial: **$5.20**

Botox concentration is calculated as:

```text
100 units ÷ diluent amount (mL)
```

Example:

```text
100 units ÷ 2.5 mL = 40 units/mL
```

---

# Volume and syringe clarification

Important:

```text
1 mL = 1 cc
```

Also:

```text
1 mL = 100 U-100 syringe markings
```

Examples:

```text
0.20 mL = 0.20 cc = 20 U-100 markings
0.30 mL = 0.30 cc = 30 U-100 markings
1.00 mL = 1.00 cc = 100 U-100 markings
```

**U-100 syringe markings are not cc.**

A syringe marking is a scale reference, not a cubic centimeter.

---

# Syringe cost assumptions

- Syringe package cost: **$18.00**
- Syringes per package: **100**
- Cost per syringe: **$0.18**

Operational default:

```text
1 syringe per 1–2 treatment areas
```

Examples:

```text
1–2 treatment areas = 1 syringe
3–4 treatment areas = 2 syringes
5–6 treatment areas = 3 syringes
```

This avoids incorrectly treating U-100 markings as Botox units.

---

# PPE assumptions

## Gloves

- Cost per glove: **$0.010**
- Default gloves per treatment session: **2**

Calculation:

```text
2 × 0.010 = $0.020
```

## Alcohol prep pads

- Cost per pad: **$0.002**
- Default estimate: **1–2 prep pads per treatment area**

Example for 3 treatment areas:

Low estimate:

```text
3 × 1 × 0.002 = $0.006
```

High estimate:

```text
3 × 2 × 0.002 = $0.012
```

# Cost allocation logic

The script allocates Botox product cost proportionally based on units used.

Example:

```text
60 units used out of 100 units = 60% of vial
$656 × 0.60 = $393.60 product cost used
```

Saline is also allocated proportionally:

```text
$5.20 × 0.60 = $3.12 saline allocation
```

Consumables are estimated separately:

```text
syringe cost + glove cost + prep pad cost
```

# Total operational vial cost

Using the default assumptions:

```text
Botox vial = $656.00
Bacteriostatic saline = $5.20
Approximate session supplies = variable by treatment plan
```

For simple break-even reference:

```text
$656.00 + $5.20 = $661.20 baseline vial cost
```

Approximate baseline break-even per unit:

```text
$661.20 ÷ 100 = $6.61/unit
```

Session-level consumables may slightly increase the effective cost.

# Pricing modes

## Standard pricing

Default target gross margin:

```text
20%
```

Formula:

```text
recommended charge = session cost ÷ (1 − target margin)
```

Example:

```text
$397.65 ÷ 0.80 = $497.06
```

---

## Family-friend pricing

Family-friend pricing uses:

```text
$10.00/unit
```

Formula:

```text
recommended charge = total planned units × $10.00
```

Example:

```text
32 units × $10.00 = $320.00
```

---

## Custom pricing

Custom pricing uses a user-provided per-unit price.

Example:

```text
$12/unit
```

Formula:

```text
recommended charge = total planned units × custom per-unit price
```
